# 💰 Personal Finance Dashboard 🎮

A fully interactive, **gamified** React-based financial dashboard to track income, expenses, explore income generation opportunities, and monitor personal development goals. Built specifically for South African developers looking to improve their financial situation **while having fun!**

## 🎮 NEW: Gamification Features!

Transform your financial journey into an exciting game:
- 🏆 **40+ Achievements** to unlock
- ⭐ **8 Levels** to progress through (Beginner → Titan)
- 🎉 **Celebration Animations** for every milestone
- 📊 **Points System** for all positive actions
- 🎯 **Progress Tracking** across 8 categories
- 💫 **Visual Feedback** with confetti and animations

**[Read Full Gamification Guide →](GAMIFICATION.md)**

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

The app will open automatically at `http://localhost:3000`

## How to Use

### Managing Expenses
1. Click "Add Expense" to create new expenses
2. Click "Edit" on any expense to modify it
3. Click "Move to Variable/Fixed" to reorganize
4. Click "Delete" to remove expenses
5. All changes save automatically

### Tracking Income Ideas
1. Choose an income idea card
2. Click "Start Tracking" or "+ Add Entry"
3. Log time spent, money invested, and money generated
4. View ROI and progress stats on the card
5. Track multiple ideas simultaneously

### Unexpected Transactions
1. Click "+ Add Transaction"
2. Select type (Income or Expense)
3. Enter description, amount, and date
4. View impact on your monthly balance

### Gym & Language Tracking
1. Click "+ Log Session"
2. Enter session details (duration, type, notes)
3. Set monthly goals
4. Track progress with visual progress bars

### Generating Reports
1. Go to "Monthly Report Generator"
2. Click "Generate PDF Report" for formatted PDF
3. Or click "Generate Text Report" for simple text file
4. Reports include all your data and recommendations

## Tips for Success

### Debt Payoff Strategy
- **Current debt**: R30,000 at ~20% interest
- **Minimum payment**: Likely R600-800/month
- **Recommended**: R2,000-3,000/month
- **Timeline**: 10-15 months to debt-free
- **Interest saved**: R3,000-5,000 by paying faster

### Quick Wins to Find Extra Money
1. **Review subscriptions**: Cancel unused services (potential R500/month)
2. **Negotiate insurance**: Shop around for better rates (potential R200/month)
3. **Meal prep**: Reduce food costs from R2,000 to R1,500 (saves R500/month)

5. **Sell unused items**: One-time boost of R2,000-5,000
